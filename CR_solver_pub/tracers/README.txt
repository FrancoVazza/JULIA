Test sequence of particle tracers data written in HDF5 format.
Each file represent a different evolutionary moment of 25 gas tracers, with quantities:
        "Redshift"  
        "xcoord"   [kpc]
        "ycoord"   [kpc]
        "zcoord"   [kpc]
        "x-velocity"   [cm/s]
        "y-velocity"   [cm/s]
        "z-velocity"   [cm/s]
        "density"  [g/cm3]
        "temperature" [K]
        "mach"
        "B"        [G]
        "div_v"    [1/s]
        "curl_v"   [1/s]
        "particleID"
